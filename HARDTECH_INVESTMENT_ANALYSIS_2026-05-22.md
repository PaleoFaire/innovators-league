# Hard-Tech Investment Analysis — Top 50 (2026-05-22)

**Why this list is different.** The first analysis skewed biotech/pharma-heavy. This one is re-scored against a strict **"important hard physical tech"** filter — companies that bend metal, build silicon, fly, generate/move energy, or sense the physical world. The following categories were **hard-excluded**:

- Pharma / therapeutics / drug-discovery / gene-editing / longevity pills (Xaira, Moonwalk, Stylus, Liberate, Kamau, Gordian, Rubedo, Pearl, Karius, Cellino, Loyal, etc.)
- Food / consumer-bio (Zero Acre, Tender Food, Minus Coffee, Leaft, Formo, Macro Oceans)
- Software / marketplace plays mis-tagged as hardware (**Eli** — IRA-incentive API; **Lumen, Lemon, InRange, Musa, Safi** — solar/recyclables marketplaces)

**Bio-hardware kept** where the product is a physical device: Paradromics (BCI implant), Somnee (EEG headband), Overture Life (IVF-lab-in-a-box).

**Pool.** Re-scored the full **568 newly-added companies** (not just the 121 I enriched). 379 survived the hard-tech filter; these are the top 50. **47 of 50 have research-verified data** (founder, stage, raise confirmed in the last 24h); 3 still need a verification pass (flagged ⚠).

**Scoring (out of 100+):** stage fit (25, seed/A best) · tier-1 VC (20) · hard-sector weight (25, defense/space/nuclear highest) · US-based (10) · data depth (10) · momentum incl. govt contracts (18) · trip-city access (12).
